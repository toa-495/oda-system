# 家計経費UI

スプレッドシート「会計記録」をデータベースとして使う、スマホ向けの簡易経費登録UIです。

## できること

- 経費一覧表示
- 新規追加
- 項目タップで詳細表示
- 編集
- 削除
- レシート撮影・アップロード
- 支払者：`マスタ!B列` の空白除外リスト
- 種別：`マスタ!C列` の空白除外リスト
- 金額：半角数字のみ
- 入力条件：支払者は必須。`タイトル＋金額` または `レシート` のどちらかが必須

## スプレッドシート前提

- データタブ：`データベース`
- 見出し：2行目
- データ開始：3行目
- 使用列：B:J

| 列 | 内容 |
|---|---|
| B | No. |
| C | タイトル |
| D | レシートURL |
| E | レシートID |
| F | 金額 |
| G | 支払者 |
| H | 種別 |
| I | 日付 |
| J | 精算 |

## 導入手順

1. スプレッドシートに紐づくApps Scriptを開く
2. `gas_api.gs` の中身を貼り付ける
3. `APP_CONFIG.RECEIPT_FOLDER_ID` に、レシート保存先DriveフォルダIDを入れる
4. Apps Scriptを「ウェブアプリ」としてデプロイする
   - 実行ユーザー：自分
   - アクセスできるユーザー：全員、またはリンクを知っている全員
5. 発行された `/exec` URLを `config.js` の `GAS_BASE_URL` に貼る
6. このフォルダ一式をGitHub Pagesにアップする

## 注意

- 削除時はNo.列を残し、C:J列をクリアします。既存の連番構造を壊さないためです。
- レシートアップロード後に保存ボタンを押すと、URLとファイルIDがスプレッドシートに記録されます。
